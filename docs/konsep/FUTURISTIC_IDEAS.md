# Ide Futuristik (Next-Gen Features)

## 1. Mekanik Inovatif
- [ ] Combo & Synergy Elements (Fire + Fire = Blaze).
- [ ] Destructible Terrain (Hancurkan dinding grid).
- [ ] Tactical Orders (Intervensi manual pemain saat pertarungan).

## 2. Kedalaman Karakter (Personality)
- [ ] Mood & Fatigue System (Butuh rotasi pahlawan).
- [ ] Hero Bonds (Sinergi persahabatan antar unit).
- [ ] Trait Evolution (Trait berubah berdasarkan pengalaman bertarung).

## 3. Dunia Dinamis
- [ ] Global Weather System (Hujan memengaruhi elemen petir/api).
- [ ] Player Outposts (Membangun fasilitas di wilayah liar).
- [ ] Bounty Board (Sayembara mengalahkan pahlawan di Leaderboard).

## 4. Stone & Crafting Lanjut
- [ ] Stone Polishing (Memproses batu kasar menjadi perhiasan indah).
- [ ] Cursed Stones (Stat tinggi dengan efek samping negatif).

## 5. Visual & UX
- [ ] Battle Ghosts (Melihat bayangan pertarungan sebelumnya di grid).
- [ ] Day/Night Cycle di background pertarungan.

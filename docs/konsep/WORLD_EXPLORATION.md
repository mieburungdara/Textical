# Konsep Dunia & Penjelajahan

## 1. Peta Wilayah (Region)
- [x] Sistem Node-based map.
- [x] Perbedaan Biome (Forest, Ruins, Town).
- [ ] World Map View: Antarmuka untuk pindah antar wilayah.
- [ ] Scouting Power: Pahlawan tipe Archer/Rogue membuka lebih banyak info pahlawan musuh.

## 2. Event & Loot
- [x] Authoritative Loot Factory (Server yang menentukan drop).
- [x] Monster Rarity (Rare monster memberikan loot lebih banyak).
- [ ] Random World Events: Menemukan peti karun atau NPC pedagang keliling.
- [ ] Global World Boss: Monster raksasa yang dilawan seluruh pemain server.

## 3. Quests
- [x] Kill Quest tracking.
- [x] Fame Score system sebagai progresi reputasi pemain.
- [ ] Chain Quests: Cerita yang berlanjut antar wilayah.
